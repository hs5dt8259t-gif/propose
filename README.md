# 結袂の環（むすびのわ）

「発見 → 鑑定 → 解放」

『結袂の環』は、RPG の未鑑定アイテム体験をモチーフに制作した、NFC × 電子ペーパー × Web 演出プロジェクトです。

プレイヤーは一つのアイテムを発見し、スマートフォンで鑑定を行い、隠された物語や実績、そしてその先に続く物語を解放していきます。

本プロジェクトは Path of Exile ２に見られるアイテム鑑定文化へのオマージュとして制作されました。

## Features

* 電子ペーパー向け UI デザイン
* Path of Exile 風アイテム表示
* NFC による Web ページ起動
* GitHub Pages による静的ホスティング
* 日本語中心のゲーム風テキスト演出
* 実物の指輪と連動した体験設計

## Story

銀の指輪：結袂の環

「袂を結ぶ」とは、同じ道を歩むこと。

鑑定によって明かされるのは性能ではなく、
その指輪に込められた記憶と物語。

今後の旅路も、
どうか手を取り合って。
